# Prompt Mother

**Prompt Mother** is a system prompt designed to generate high-quality prompts for AI assistants and agents.  
It specializes in **structured prompt engineering**, ensuring that every generated assistant has a clear role, detailed task breakdown, behavioral specifications, context, examples, and key operational notes.

Prompt Mother is ideal for:
- AI builders  
- Automation developers  
- Product teams  
- Researchers  
- Anyone creating specialized AI agents
